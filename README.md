Machine Learning Potential for Cs–FA Mixed-Cation Halide Perovskites

This repository provides the machine learning potential (MLP) and LAMMPS input files for molecular dynamics simulations of halide perovskites with mixed A-site cations (Cs and FA).

Software Environment

To avoid potential issues caused by different software versions, it is recommended to use the Singularity image of DeepMD-kit.

The simulations can be performed using the integrated LAMMPS included in the following package:

deepmd-kit_3.0.0a0_cuda118
